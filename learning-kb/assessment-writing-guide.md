# Assessment Question Writing Guide (Synthetic)

> Synthetic demonstration document from the internal Learning & Development Playbook.
> Used by the Assessment Agent to generate credible, grounded practice questions.

## Question quality rules
1. Every question must be answerable from the approved study materials — cite the
   source document the question derives from
2. Scenario-first phrasing: describe a realistic situation, then ask what to do —
   avoid pure definition recall
3. One clearly best answer; distractors must be plausible actions a practitioner
   might actually consider, not obvious throwaways
4. Match the domain weighting of the certification: a five-question set for a
   certification where one domain carries 30% should put at least one (usually two)
   questions in that domain
5. Target the learner's weak domains: at least half the questions should probe
   domains self-rated 3 or below

## Difficulty calibration
- Readiness checks should sit slightly above expected exam difficulty — learners who
  clear a harder practice bar pass more reliably
- A learner scoring 4/5 or better on a weighted, weak-domain-focused set is a strong
  booking signal alongside the 75% practice threshold

## Evaluation write-up format
For each assessment, record: questions asked (with citations), domains covered,
score interpretation against the pass threshold, and one specific next action
(book / focused revision on domain X / extend plan by N weeks).
